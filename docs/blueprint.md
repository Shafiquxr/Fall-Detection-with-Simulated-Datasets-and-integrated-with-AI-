# **App Name**: Guardian Angel

## Core Features:

- Caregiver Configuration: Configurable caregiver list, ordered by priority and including contact methods (SMS, call, app notification).
- Fall Simulation: Simulate fall detection based on sensor data, using pre-defined thresholds and a timer to minimize false positives.
- Alert Escalation Logic: Use tool that determines when alerts are to be escalated to the next caregiver based on response time from the previous one.
- Alert Notification: Mobile application for caregivers that displays received alerts.
- Location display: Display information from the wearable (i.e., location) on a map within the app.

## Style Guidelines:

- Primary color: Soft blue (#77B5FE), evoking a sense of calmness and safety.
- Background color: Light gray (#F0F4F8), offering a clean and unobtrusive backdrop.
- Accent color: Gentle green (#82E0AA) to signal all-clear states and successful actions.
- Font pairing: 'Inter' (sans-serif) for body text and headlines.
- Use clear, simple icons to represent actions and alert types.
- Prioritize easy-to-read alerts and simple app navigation with intuitive placement.
- Subtle animations to indicate the status of an alert and acknowledge actions.
